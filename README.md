tkinterを使って、遺伝的アルゴリズム（Genetic ALgorithm, GA）で単制約0-1ナップザック問題（Knapsack Problem, KP）を解いた時の世代ごとの最適値の変化を可視化するGUIを作ってみました。
遺伝的アルゴリズムやナップザック問題については他サイトを参考にしてください。

# 使用方法
使用言語はpythonです。実行すると、画面中央に数値入力が可能な画面が現れます。ここでは、GAとKPの設定を行うことができます。

# 設定可能項目

**GA Parameter**
- Runs: GAの実行回数
- Items Size: ナップザック問題で扱う品物の数。これは遺伝子長でもあります。
- Poplation Size: GAの母集団サイズ
- Generations: GAの最大世代数
- Crossover Rate: 交叉率。ここでは、GAは一点交叉のみが実装されています。
- Mutation Rate: 突然変異率。ここでは、Bitflip変異のみが実装されています。
- Capacity: ナップザック問題の容量制約

**Data Source**
- v_min: 生成する品物の価値の最小値
- v_max: 生成する品物の価値の最大値
- w_min: 生成する品物の重さの最小値
- w_max: 生成する品物の重さの最大値

- 自動生成: 自動生成では上記のData Sourceの設定に従い、Item Size個の品物を一様分布に従って生成する。
- ファイルから読み込み: 価値と重さが書かれたtxtファイルを渡すことで、その問題をGAで解くことができます。読み込ませるデータの形式はdataset_1.txtらを参考にしてください。

# 表示項目

**Knapsack Data**
- Values: GAで解いたナップザック問題の各品物の価値が表示されます
- Weights: GAで解いたナップザック問題の各品物の重さが表示されます

**Result**
結果がGA Progressとしてグラフで表示されます。縦軸が品物の総価値、横軸がGAの世代数です。

Resultの枠では与えられた容量制約下での総価値と総重量がグラフの下に表示されます。

# 実行ボタン
実行ボタンはResultの下にあります。このボタンをクリックすると、プログラムが実行され、与えられたナップザック問題をGAで解き、得られた最良値がResultに表示されます。実行ボタンをクリックするたびに結果が変化します。

# 終了ボタン
終了は画面左上隅のボタンか、実行ボタンの下にある赤い字で終了と書かれたボタンをクリックすると実行を終了します。右上隅のXマークで終了すると、正常に実行が終了しないので注意してください。
